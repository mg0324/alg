## 递归序

![](https://mg.meiflower.top/oss/alg/ds/tree/tree2.png)

树的遍历，也称之为获取树的递归序。
```
1->2->4->4->4->2->5->5->5->2->1->3->6->6->6->3->7->7->7->3->1
```

重点：每一个节点都有三次机会到达自己。

## 树的遍历
* 前序遍历（根，左，右），也叫深度优先遍历

```
# 递归序里每个节点第一次到自己时就打印
1->2->4->5->3->6->7 
```

* 中序遍历（左，根，右）（二叉搜索树，左<根<右）

```
# 递归序里每个节点第二次到自己时就打印
4->2->5->1->6->3->7
```

* 后序遍历（左，右，根）（LC230)

```
# 递归序里每个节点三次到自己时就打印
4->5->2->6->7->3->1
```

* 广度优先遍历，也叫宽度优先遍历

```
宽度优先遍历：(wfs) 队列顺序
1->2->3->4->5->6->7->
```
